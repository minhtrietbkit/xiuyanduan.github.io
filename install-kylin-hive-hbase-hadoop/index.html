install kylin hive hbase hadoop

Posted on 2017-01-20   |  

本文主要记录了安装hadoop，hive，zookeeper，hbase，kylin的整个过程，其中各软件的版本如下：

  • hadoop-2.7.3.tar.gz
  • apache-hive-2.1.1-bin.tar.gz
  • hbase-1.2.4-bin.tar.gz
  • zookeeper-3.4.9.tar.gz
  • apache-hive-2.1.1-bin.tar.gz

直接在各官网下载较慢，可使用中国互联网络信息中心开源镜寻找各软件

  1. 操作系统使用的为Ubuntu 14.04.4 LTS，java version为1.8.0_111
  2. MySQL使用操作系统自带的tasksel安装，版本号为5.5.53-0ubuntu0.14.04.1
  3. 为避免权限问题，以下全部使用root用户执行
  4. 各软件包均下载至/usr/local/src，为简便起见，均解压至/opt，例如/opt/hadoop-2.7.3
  5. 本文提到的服务器hostname为master.xyduan.com，也通过域名解析到master.xyduan.com，ssh生成的秘钥对已将公钥加入自己的authorized_keys，以下截屏涉及到的显示为root@master由PS1值默认设置决定
